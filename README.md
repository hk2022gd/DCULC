# DCULC 智能卡读写器

一款现代化的接触式智能卡读写工具，支持多种常见的逻辑加密卡和存储卡。

## ✨ 功能特性

### 读写操作
- **整卡读取** - 根据卡片类型自动识别容量，一键读取全部数据
- **数据写入** - 支持修改后的数据回写到卡片
- **HEX/ASCII显示** - 直观的十六进制数据查看和编辑界面

### 卡片管理
- **自动识别** - 自动检测插入的卡片类型
- **手动选择** - 支持手动指定卡片类型
- **密码验证** - 支持 SLE4442/4428 等加密卡的密码验证
- **密码修改** - 验证成功后可修改卡片密码
- **密码计数器** - 显示剩余密码尝试次数

### 文件操作
- **保存 Dump** - 将卡片数据保存为本地文件
- **载入 Dump** - 从文件加载数据进行比对或写入

### 其他
- **现代UI** - 采用暗色主题和毛玻璃效果的精美界面
- **自动关闭** - 关闭浏览器标签页时自动停止后台服务
- **便携打包** - 可打包为独立 EXE 运行

## 📋 支持的卡片类型

| 卡片类型 | 容量 | 密码保护 |
|---------|------|----------|
| SLE4442/4432 | 256 字节 | ✅ 3字节密码 |
| SLE4428/4418 | 1024 字节 | ✅ 2字节密码 |
| AT24C01 | 128 字节 | ❌ |
| AT24C02 | 256 字节 | ❌ |
| AT24C04 | 512 字节 | ❌ |
| AT24C08 | 1024 字节 | ❌ |
| AT24C16 | 2048 字节 | ❌ |
| AT24C64 | 8192 字节 | ❌ |

## 🔧 系统要求

- **操作系统**: Windows 7/10/11
- **Python**: 3.x (32位版本，用于开发/构建)
- **读卡器**: DCULC 系列读卡器 (RD600-ULC / D6-ULC 等)

## 🚀 快速开始

### 方式一：使用打包好的 EXE（推荐）

1. 运行 `build.bat` 构建可执行文件
2. 在 `dist` 目录下找到 `DCULC_Reader.exe`
3. 双击运行，浏览器会自动打开应用界面

### 方式二：开发模式运行

```bash
# 安装依赖
pip install flask flask-cors waitress pyinstaller

# 运行后端
cd DCULC_Modern_App/backend
python app.py
```

## 📖 使用说明

1. **连接设备** - 点击"连接读卡器"按钮
2. **识别卡片** - 插入卡片后点击"自动识别"
3. **密码验证** - 如果是加密卡，输入密码并验证
4. **读取数据** - 点击"整卡读取"获取卡片数据
5. **编辑数据** - 在界面中直接编辑十六进制数据
6. **写入修改** - 点击"写入修改"将更改保存到卡片


## ⚠️ 注意事项

- **密码尝试次数有限**：SLE4442 只有 3 次，SLE4428 只有 8 次，用完后卡片将永久锁定
- **必须使用 32 位 Python**：DLL 为 32 位，需要匹配的 Python 环境
- **保持卡片稳定**：读写过程中请勿移动卡片
